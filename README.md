# Hotel Management System

A web-based **Hotel Management System** (HMS) built using **Django** and **Tailwind CSS**, designed to help hotel owners (property listeners) manage their hotel listings, bookings, and room availability, while offering customers an easy and responsive platform for booking rooms.

## Features:
- **User Roles**: 
  - **Admin**: Full control over the system (hotel approval, booking management, and user monitoring).
  - **Property Listener**: Manage hotel properties, rooms, and bookings.
  - **Customer**: Search and book hotel rooms based on filters (location, price, availability, etc.).

- **Core Features**:
  - **Hotel & Room Management**: Add multiple hotels with room types, prices, and availability.
  - **Search & Filter**: Find hotels based on location, price range, and available rooms.
  - **Booking System**: Customers can book rooms with check-in/check-out dates and monitor booking status (Pending/Confirmed/Cancelled).
  - **Responsive Design**: Clean, responsive UI built with **Tailwind CSS** for desktop, tablet, and mobile users.

## Tech Stack
- Backend: Django
- Frontend: HTML, Tailwind CSS, JavaScript
- Database: SQLite (can be upgraded to PostgreSQL)
- Authentication: Django Auth
